## Clawe CLI

```bash
clawe check agent:${AGENT_ID}:main           # Check notifications
clawe tasks agent:${AGENT_ID}:main           # List my tasks
clawe task:create "title" --assign <key>     # Create & assign task
clawe task:view <taskId>                     # View task details
clawe squad                                  # Squad status
clawe feed                                   # Activity feed
```
